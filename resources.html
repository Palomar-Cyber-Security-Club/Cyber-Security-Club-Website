Learning Resources

Curated links to help you grow your cybersecurity skills

Start Here — Beginner Paths
Free / Beginner

TryHackMe — Pre-Security Path

Zero experience needed. Covers networking fundamentals, Linux basics, and web fundamentals before touching any hacking.

Free / Beginner

Cybrary

Free cybersecurity training videos and courses. Great for structured learning on specific certifications and topics.

Free / Beginner

Professor Messer

Free CompTIA study materials including video courses for Security+, Network+, and A+. Highly recommended for cert prep.

Free / Beginner

OverTheWire — Bandit

Linux command-line wargame for absolute beginners. Learn terminal skills through a series of escalating challenges.

Certifications
Entry Level

CompTIA Security+

The most widely recognized entry-level security cert. Often required for government and DoD positions. Great starting point.

Networking

CompTIA Network+

Validates networking knowledge and skills. Understanding networks is essential for cybersecurity — do this before Security+.

Advanced Security

CompTIA CASP+

CompTIA Advanced Security Practitioner — an advanced-level cert for security architects and engineers. DoD 8570 compliant.

Networking — Cisco

CCNA

Cisco Certified Network Associate. Industry-recognized networking cert covering routing, switching, and network fundamentals.

ISC2 — Entry

ISC2 Certified in Cybersecurity (CC)

Free entry-level cert from ISC2. Great first certification with no experience required — includes free training and exam voucher.

ISC2 — Advanced

ISC2 CISSP

Certified Information Systems Security Professional — the gold standard for senior security professionals. Requires 5 years of experience.

ISC2 — Cloud

ISC2 CCSP

Certified Cloud Security Professional. Validates expertise in cloud security architecture, design, operations, and compliance.

Penetration Testing

OSCP — OffSec

The gold standard penetration testing certification. Hands-on, grueling, and highly respected in the industry.

Cloud Security

AWS Security Specialty

Cloud security is in massive demand. This cert validates your ability to secure AWS workloads and infrastructure.

Reading & Reference
Reference

OWASP Top 10

The definitive list of the most critical web application security risks. Required reading for anyone doing web security.

News

The Hacker News

Stay current on vulnerabilities, breaches, and cybersecurity news. Updated daily with the latest threats and exploits.

CVE Database

NIST NVD

The National Vulnerability Database. Search and track CVEs, vulnerability severity scores, and patch information.

Cheat Sheets

OWASP Cheat Sheet Series

Concise, actionable security guidance on specific topics. Bookmark this — you'll reference it constantly.

David Bombal

Networking and ethical hacking content. Interviews with industry professionals and hands-on lab demonstrations.

Cyber Maddy

Cybersecurity content covering ethical hacking, CTF walkthroughs, and career advice for aspiring security professionals.

YouTube

Practical cybersecurity content from a working security specialist. Real-world tips, tools, and career guidance.

Community & Career
Community

r/netsec

Technical information security community on Reddit. Quality discussions, research papers, and news from practitioners.

Job Board

Cybersecurity career pathway tool and job market data. See what roles exist, what they pay, and what certs are needed.
